KiwiDocs
The open-source documentation system.
Made for GitHub
KiwiDocs was designed to fit well with GitHub, because we know that a lot of developers uses GitHub to share their works. We designed it to work with GitHub Pages, and GitHub Wiki.
Markdown Powered
Save time and focus on your project's documentation. Simply write docs posts with Markdown and KiwiDocs will handle the rest. This will help you stay productive, even if you don't use GitHub.
Minimalist
We made KiwiDocs as light as possible, to help you and your users focus on the content and not on the design. Our goal is to provide the most minimalistic interface as possible.
Customizable
Add your logo, your menus, your titles and your content through config.json
